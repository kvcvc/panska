# Studentský portál

Monorepozitář studentského portálu SPSŠT Panská

## Spuštění

1. Vytvořte lokální kopii repozitáře:
   > git clone https://github.com/kvcvc/studentsky-portal.git
2. Nainstalujte závislosti:
   > yarn install
3. Spusťte prostředí:
   > yarn start
